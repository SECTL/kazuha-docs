# Kazuha-docs

这是 [Kazuha](https://github.com/SECTL/kazuha) 文档仓库，使用 VitePress 框架

## 安装

安装依赖：

```bash
npm install
```

### 开发

启动开发服务器：

```bash
npm run dev
```

### 构建项目

构建生产版本：

```bash
npm run build
```

预览生产产物：

```bash
npm run preview
```
